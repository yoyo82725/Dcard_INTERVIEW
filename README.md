# Dcard_INTERVIEW

2017 Senior Web Frontend Developer

1. Todo List
使用任意 JavaScript library 或 framework，實作一個 todo list，至少必須包含以下功
能：
• 新增 todo
• 刪除 todo
• 把 todo 標記為已完成 / 未完成
• 修改 todo 內容
• 過濾已完成 / 未完成的 todos
• 清除已完成的 todos
• 加分：RWD, 測試

2. Photo Uploader
Dcard 每天都有大量使用者上傳圖片，請設計一組後端服務及前端元件：
• 使用者可以選擇本地檔案或 url 上傳圖片
• 可以裁切 (crop) 圖片
• 可以簡單處理圖片，譬如在指定位置馬賽克
• 至少支援 jpeg, png 圖片格式
• 服務可以儲存圖片
• 服務可以依需求回傳指定尺寸的圖片縮圖
• 其他任何你可以想得到的強化、優化功能
